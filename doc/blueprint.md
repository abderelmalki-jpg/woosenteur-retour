# **App Name**: ForgeParfum IA

## Core Features:

- Génération de Descriptions SEO: Utilise l'IA Gemini pour générer des descriptions de produits optimisées pour le SEO, basées sur les détails fournis par l'utilisateur.
- Saisie d'Informations Produit: Permet aux utilisateurs de saisir le nom du produit, la marque, l'audience cible, les notes principales et l'ambiance pour les parfums.
- Gestion des Crédits: Suit et gère les crédits des utilisateurs pour la génération IA, en décrémentant les crédits après chaque génération de description réussie.
- Sauvegarde des Produits dans Firestore: Enregistre les descriptions de produits générées, ainsi que les métadonnées associées, dans une base de données Firestore pour un suivi historique.
- Suivi de l'Historique: Affiche les 10 derniers produits générés, récupérés directement depuis Firestore.
- Export WooCommerce: Outil pour exporter les fiches produits directement vers une boutique WooCommerce via l'API WooCommerce.

## Style Guidelines:

- Couleur primaire: Rouge amarante riche (#C1292E). Évoque un sentiment de luxe, de chaleur et de richesse olfactive, pertinent pour les parfums. Convient comme couleur frappante sur un thème clair.
- Couleur de fond: Rose blush doux (#F8E7EB). Une version très légère et désaturée du primaire qui fournit une toile de fond subtile et élégante, améliorant la lisibilité.
- Couleur d'accent: Abricot chaud (#F46036). Cette couleur, légèrement décalée sur la roue chromatique et rendue plus vibrante, attire l'attention sur les éléments interactifs clés sans surcharger l'esthétique globale.
- Corps: 'PT Sans', une police sans-serif polyvalente et humaniste adaptée aux principaux éléments de l'interface utilisateur et aux blocs de texte.
- Titre: 'Playfair', une police serif à contraste élevé, idéale pour attirer l'attention, donnant une impression élégante et à la mode aux principaux en-têtes.
- Utiliser des icônes minimalistes de Lucide-react.
- Mettre en œuvre une mise en page claire et conviviale qui s'aligne sur les meilleures pratiques SEO pour les descriptions de produits. Envisager une structure hiérarchique et un flux logique pour améliorer l'expérience utilisateur.
